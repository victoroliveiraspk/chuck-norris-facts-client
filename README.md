# Chuck Norris facts client

## Technologies that was used to build this app:

```
Next.JS
Typescript
Tailwindcss
Cypress
GraphQL
```

## How to set up:

### To make local server, run:

```
yarn dev
```

### To make the build direcotry, run:

```
yarn build
```

### To make local build server, run:

```
yarn build
yarn start
```

### To run tests:

```
yarn cypress:open
